# VntGo
A super simple theme for Hugo.

## screenshot
![Index Screenshot](screenshot/index.png)

## Installation

### Installing this theme
```
$ cd themes/
$ git clone https://github.com/xwyc/hugo-theme-vntgo
```

### Specify vntgo as your default theme in the config.toml file
```
theme = "vntgo"
```
